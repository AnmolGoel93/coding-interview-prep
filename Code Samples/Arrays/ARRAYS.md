### Time Complexity

Method       | Time       | Other Info.
:----------- | :--------: | ----------:
sorted()     | O(nlog(n)) | Uses Tim sort
list.sort    | O(nlog(n)) | Uses Tim sort
max(), min() | O(n)       | Use if condition, if possible than these
in           | O(n)
len()        | O(1)       | Avoid using this inside a loop rather use local var
